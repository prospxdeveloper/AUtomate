# Icon Placeholder

The icons need to be actual PNG files. For now, you can:

1. Use an online SVG to PNG converter to convert icon.svg
2. Create the following files:
   - icon-16.png (16x16)
   - icon-48.png (48x48)
   - icon-128.png (128x128)

Or use this temporary workaround:
Copy any 16x16, 48x48, and 128x128 PNG images into this folder with the correct names.

The extension will work without icons, but they won't display properly.
